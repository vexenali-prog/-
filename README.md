# Paper Trading Bot

Торговый бот для крипты (BTC, ETH, SOL). Сейчас работает в **paper-режиме**:
торгует виртуальными деньгами по реальным ценам, чтобы доказать (или
опровергнуть) прибыльность стратегии до подключения реального счёта.

## Как это работает

- Каждый час GitHub Actions запускает `bot/run.py`
- Бот берёт свежие часовые свечи с публичного API OKX (цены совпадают с
  Bybit; API Bybit блокирует серверы GitHub, поэтому данные с OKX)
- Стратегия: следование за трендом
  - вход: цена закрепилась выше EMA(700) + 2%
  - выход: цена ушла ниже EMA(700) − 2%
  - защита: стоп-лосс −10% от входа, трейлинг-стоп −20% от пика,
    BTC-фильтр (покупки только когда биткоин выше своей EMA), фильтр
    объёма (вход только при объёме >1.5× среднего за 20 дней); после
    стоп-выхода повторный вход — только после «перезарядки» сигнала
  - пирамидинг: при +15% от входа докупается половина доли (раз на
    позицию, вход усредняется) — проверен свипом параметров и
    walk-forward на обеих половинах истории
  - каждой монете — равная доля капитала, комиссия 0.1% за сторону учтена
- Торгуются 10 инструментов: ядро из крупнейших (BTC, ETH, SOL, XRP,
  DOGE, TRX) плюс SUI, ARB, XLM и золото XAUT — отобраны скринингом
  десятков ликвидных кандидатов
  (каждая в одиночку обыгрывает buy-and-hold, корзина из 9 на общем
  2-летнем окне даёт +116% против +82% у шестёрки при той же просадке).
  Золото (токен XAUT) добавлено 21.07: в корзине улучшает и доходность,
  и просадку в обеих половинах истории — диверсификация вне крипты.
  Ещё 25 активов — в наблюдении: видны в меню, но не торгуются (шумные
  альты на истории сливают трендовую стратегию: SHIB -33%, OP -50%,
  TIA -64% и т.д.).
- Почему так: на бэктесте ядро портфеля за 4.2 года (обвал 2022, крах
  FTX, бычий 2024) даёт **+191%** при просадке −19%; корзина из 9 монет
  за 2 года — **+117%**. Отвергнуты проверкой на тех же данных: RSI
  30/70 (−34%), свечные паттерны (−71%), сеточные боты, тейк-профиты,
  отскоки от поддержки, «покупай при страхе» — подробности в истории
  коммитов.

## Интерактивное меню (worker/)

Кнопки в Telegram обслуживает Cloudflare Worker (`worker/worker.js`):
отвечает мгновенно, цены берёт с биржи в момент нажатия, портфель — из
`state/paper_state.json` этого репозитория. Деплой: `wrangler deploy` из
каталога `worker/`, секреты `BOT_TOKEN` и `WEBHOOK_SECRET` через
`wrangler secret put`.
- Портфель хранится в `state/paper_state.json` (коммитится обратно в репо)
- Сделки и дневная сводка приходят в Telegram

## Настройка (один раз)

1. Создай бота: напиши [@BotFather](https://t.me/BotFather) → `/newbot` →
   скопируй токен.
2. Напиши своему новому боту любое сообщение, затем узнай свой chat id:
   открой `https://api.telegram.org/bot<ТОКЕН>/getUpdates` и найди
   `"chat":{"id":...}`.
3. В репозитории: Settings → Secrets and variables → Actions → New repository
   secret. Добавь два секрета:
   - `TELEGRAM_BOT_TOKEN` — токен из шага 1
   - `TELEGRAM_CHAT_ID` — id из шага 2
4. Вкладка Actions → включи workflows, если GitHub спросит. Можно нажать
   «Run workflow» у «Paper trading bot» для проверки.

Без секретов бот тоже работает — просто пишет сообщения в лог Actions
вместо Telegram.

## Команды локально

```bash
python -m bot.run             # один шаг paper-трейдинга
python -m bot.backtest 9000   # бэктест на N последних часов
```

## Честные ожидания

Бэктест на прошлом не гарантирует прибыль в будущем. План такой:

1. **Сейчас** — paper-режим минимум 3–4 недели.
2. **Потом** — смотрим статистику: если бот стабильно в плюсе после
   комиссий, подключаем реальный счёт Bybit с маленькой суммой.
3. Реальная торговля добавляется отдельно (API-ключи Bybit, жёсткие лимиты
   на размер сделки). GitHub Actions для этого не подойдёт — Bybit блокирует
   IP-адреса США, понадобится дешёвый VPS в подходящем регионе.

⚠️ Криптовалюты волатильны. Не вкладывай деньги, потерю которых не можешь
себе позволить.
